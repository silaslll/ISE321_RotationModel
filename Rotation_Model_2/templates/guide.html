- How to use:

Detailed Guidance

  1. Successively click "Resident," "Rotation", and "Block" to add inputs first
    These will be Prerequisites for further inputs
  2. The right-side table is to display all recorded data. You can delete and update all inputs
  3. Once finish adding the info above, add assignment sets one by one.
  4. Click "Run Model" in the navigator to run the rotation scheduling model
  5. If error messages pop out, delete conflicts in input interfaces, and try run model again.
  6. If there's no conflicts, a rotation schedule will display in the window.
